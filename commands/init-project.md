---

`project-interview` 스킬로 새 프로젝트를 초기화한 뒤 **가이드 모드**로 진행하라.

절차:
1. 시니어 서비스 전략가로서 한국어로 인사하고, 짧은 인터뷰(4라운드)를 한다고 안내.
2. `project-interview` 스킬 순서: AskUserQuestion 4라운드 → project-brief.json → scaffold 실행 → 산출물 도구 제안 → 한국어 요약.
3. 인터뷰 없이 파일부터 만들지 말 것.
4. **가이드 모드 핸드오프**: 골격 생성 요약 후 AskUserQuestion으로 "2단계 설계로 갈까요?" [예 / 수정 / 멈춤].
   - **예** → `/next-stage`의 가이드 루프대로 2단계부터 자동 진행(각 단계 끝에 요약+확인, 3단계 리뷰 게이트는 하드 스톱).
   - **멈춤** → 여기서 종료(나중에 `/next-stage`로 재개).

추가 지시가 있으면 반영: $ARGUMENTS
